How I made this website

I started by going over varius software to use and idealy find something that was atleast similar to the look and feel i wanted from a website. I found several useful videos that guided me to the software Visual Studio Code. The varius videos i followed were:

  • How to Create & Host a Portfolio in 10 minutes with Github pages!
  • Git and GitHub for Beginners - Crash Course
  • How To Use GitHub with VS Code in 2020 | Commit & Push | Part 1
  • How to use Git inside of VSCode - 2020

After finding sutable tools I set my sites on a Website template. The website template had to be clean and easy to use, you had to be able to find exactly what you needed to find form the main page and I had tinkerd with a search function but I deamed it exesive and unessesary. Going over the template link at HTML5 UP I found "Massively". With my template found the real work began.

The template was downloaded and extracted into the site file location. that file was then opend with VS code (Visual Studio Code). Not only was "Massively" what I wanted but had a file labeled "elements" which held an asortment of useful terms and tools to use in the comming articles. VS code was then connected to my personal github page to be able to push my changes straight to the website. I made a simple reposotory on git, New > check .gitnore and Readme > create repository The template is then connected to the main branch where I can from there push my current progress to github with a push of a button on the lower left side of the screen

And from github I pulled it to the master. Suddenly i have a website. (note to spellscheck, extenction?)
