Looked simple enough, it wasnt. This became a 14 piece incased model stand. without all the minutiae the prosses is simple and easy to repeat.

Order of operation

  1. set every parameter you can
  2. set a special kerf peramiter accounting for the diameter of the laser
  3. Make the intended product
  4. lay it flat with either the align function or the nest function
  5. make a single sketch of the entire product and use the offset function to account for kerf
  6. export that sketch as a dxf file
  7. open it in inkscape and set it up for cutting
  8. In case of duplication delete one of the dupes
  9. make the line thickness 0.02mm in the outline setting
  10. save the file as a pdf and optionaly add borders to it
  11. The laser cutter must by setup and set the start point
  12. start cutting
  13. the components will need to be gently hammerd in to place and volia
  14. Enjoy the product of your labor
